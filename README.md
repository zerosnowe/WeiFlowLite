# WeiFlowLite

一款适用于Windows10Mobile设备的UWP第三方微博客户端

# 本项目仅供学习与研究使用，请在24小时内删除

功能列表

| 功能模块   | 微博 H5 能力         | WeiFlowLite 当前状态 | 缺口/说明                                 | 优先级                           |
| ------ | ---------------- | ---------------- | ------------------------------------- | ----------------------------- |
| 推荐首页   | 推荐时间线、分类切换、分页    | 已实现              | 使用 `container/getIndex`，支持分类和分页       | -                             |
| 关注时间线  | 登录后查看关注内容、分页     | 已实现              | 使用 `feed/friends`，包含 Cookie 刷新逻辑      | -                             |
| 热搜     | 实时热搜、热榜标签        | 已实现              | 使用 `weibo.com/ajax/statuses/hot_band` | -                             |
| 搜索     | 关键词搜索、分页、结果详情    | 已实现              | 使用 `container/getIndex` 搜索模式          | -                             |
| 微博正文   | 正文、图片、视频、来源、统计信息 | 已实现              | 支持独立正文页和媒体预览                          | -                             |
| 评论查看   | 热门评论、分页加载        | 已实现              | 使用 `comments/hotflow`                 | -                             |
| 发表评论   | 文本评论、表情          | 已实现              | 使用 `api/comments/create`，支持 Emoji     | -                             |
| 评论回复   | 回复某条评论、楼中楼       | 缺失               | 当前只能发表评论，不能指定 `comment_id` 回复         | 高                             |
| 评论点赞   | 点赞评论、取消点赞        | 缺失               | 没有评论态度接口和对应 UI                        | 中                             |
| 微博点赞   | 点赞、取消点赞、状态同步     | 部分实现             | 已支持正文点赞，但列表页没有交互入口，状态依赖接口返回           | 中                             |
| 关注用户   | 关注、取消关注          | 已实现              | 使用 `friendships/create/destroy`       | -                             |
| 转发微博   | 转发、带评论转发、转发列表    | 缺失               | 当前只有 `reposts_count` 展示，没有转发接口        | 高                             |
| 收藏微博   | 服务端收藏、取消收藏、收藏列表  | 部分实现             | 当前收藏只是本地 UI 状态，没有微博收藏 API，也不会跨设备保存    | 高                             |
| 用户主页   | 用户资料、微博列表、关注状态   | 已实现              | 支持头像进入主页和用户时间线                        | -                             |
| 关注列表   | 查看某用户关注的人        | 缺失               | 用户资料只展示关注数，没有关注列表页面                   | 中                             |
| 粉丝列表   | 查看某用户粉丝          | 缺失               | 没有粉丝列表接口和 UI                          | 中                             |
| 私信     | 私信列表、聊天页面        | 部分实现             | 当前通过 WebView 打开 H5 私信页，不是原生 API 实现    | 中                             |
| 通知     | @我、评论、赞、系统通知     | 缺失               | 没有通知中心接口和入口                           | 高                             |
| 登录     | 微博网页登录、Cookie 保存 | 已实现              | WebView 登录并保存移动站 Cookie               | -                             |
| 当前用户   | 登录后原生个人中心        | 已实现              | 使用配置接口获取当前用户                          | -                             |
| 退出登录   | 清理 Cookie 和本地状态  | 已实现              | 已实现本地 Cookie 清理                       | -                             |
| 发布微博   | 发文字微博            | 缺失               | 没有发布微博接口和页面                           | 高                             |
| 图片上传   | 发布图片微博           | 缺失               | 没有图片上传、压缩和上传凭证流程                      | 高                             |
| 视频上传   | 发布视频微博           | 缺失               | 没有视频上传和发布流程                           | 高                             |
| 草稿箱    | 保存、编辑、发布草稿       | 缺失               | 没有本地或服务端草稿功能                          | 中                             |
| 媒体查看   | 原图、视频播放、下载、分享    | 已实现              | 支持系统浏览器下载和系统分享                        | -                             |
| 个人资料编辑 | 修改头像、昵称、简介       | 缺失               | 当前个人中心只读                              | 中                             |
| 账号安全   | 账号管理、绑定、隐私设置     | 缺失               | 设置页目前主要是主题设置                          | 中                             |
| 举报/屏蔽  | 举报微博、屏蔽用户、拉黑     | 缺失               | 没有相关接口和 UI                            | 中                             |
| 话题/超话  | 话题页、超话主页、关注超话    | 缺失               | 搜索只能返回微博结果，没有话题实体页                    | 中                             |
| 直播/故事  | 直播、故事、短视频流       | 缺失               | 当前只处理微博图片和普通视频媒体                      | 低                             |
| 访客会话   | 未登录访问公开内容        | 已实现              | 使用 `visitor/genvisitor2` 建立匿名会话       | - |
